# Atom starter

> https://atom.io

:rocket: My favorites plugins, themes and configurations for the Atom editor. 
 
## How to use?

```sh
# Clone the project
git clone git@github.com:jjaderg/atom-starter.git ~/.atom-starter

# Access folder
cd .atom-starter

# Install preferences
apm install --packages-file packages.cson
```
